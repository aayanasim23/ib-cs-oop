// API KEY SETUP

The AI grading feature requires a free API key. Takes 2 minutes. No credit card needed for the free options.

Your key is stored only in your browser (localStorage) and your personal Supabase account — never shared, never logged.
Groq FREE — recommended

Fastest inference of any free provider. 1000 requests/day. No credit card required.

  • Go to console.groq.com
  • Sign up with Google or email
  • Click API Keys in the left sidebar
  • Click Create API Key → give it any name → copy it
  • Paste into the app settings, select Groq, hit SAVE
